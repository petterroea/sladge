# Sladge

The sladge is a badge for self-declaration of vibe coding, to be used in git repositories, package registry descriptions, and other places where the software is presented to users.

The goal of the sladge is to protect third party users from wasting time reviewing or maintaining low quality LLM-generated code, when the intention is not that humans are to do so.

The HTML for sladge.net was itself vibe coded, and text rewritten by human.

See the website for more details.
